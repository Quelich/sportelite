# Sport Elite App

Sport Elite App is a Flutter application that allows users to track their workouts and progress.
Workout coaches can create and share their workouts with their clients by using messaging service in Sport Elite App.

## Demo

You can login with the following services:

- Email
- Google
- Facebook
